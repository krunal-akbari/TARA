# Clients Module Docs

## Client Endpoints
- `create.md` - Create a client
- `list.md` - List clients
- `get.md` - Get a single client
- `update.md` - Update a client
- `delete.md` - Soft-delete a client
- `restore.md` - Restore a soft-deleted client

## Client Contact Endpoints
- `contacts-create.md` - Create a contact for a client
- `contacts-list.md` - List contacts for a client
- `contacts-update.md` - Update a client contact
- `contacts-delete.md` - Delete a client contact
